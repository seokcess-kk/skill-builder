# 네이버 블로그 SEO 텍스트 작성 가이드

## 기본 원칙

네이버 블로그 상위노출을 위한 텍스트 원고 작성 규칙입니다.
상위글 분석 결과와 병행하여 사용합니다.

---

## 1. 글 구조

### 정보성 글 구조 (기본값)
객관적 정보 전달, 전문 지식 공유 목적. 해요체/격식체 사용.
```
도입부 (100~200자)
  → 문제 제기 / 독자 공감

본론 1: 원인/배경 (300~500자)
  → 왜 이런 문제가 생기는지 설명
  → [이미지 삽입]

본론 2: 해결 방법 (400~600자)
  → 구체적 방법/솔루션 제시
  → [이미지 삽입]

본론 3: 상세 정보/비교 (300~500자)
  → 추가 팁, 주의사항, 비교 정보
  → [이미지 삽입]

정리/요약 (100~200자)
  → 핵심 포인트 정리
  → [이미지 삽입]

마무리 (50~100자)
  → CTA 또는 한줄 요약
```

> **주의**: 후기성 글은 허위/과장 광고 리스크가 있어 생성하지 않습니다. 정보성 글만 작성합니다.

### 글자수 기준
- **최소**: 1,500자 (이하이면 상위노출 어려움)
- **최적**: 2,000~3,000자
- **최대**: 4,000자 (너무 길면 이탈률 증가)
- 상위글 분석 결과의 평균 글자수에 맞추는 것이 가장 효과적

---

## 2. 키워드 삽입

### 삽입 위치 (우선순위)
1. **제목** — 메인 키워드 필수 포함
2. **첫 문단** — 도입부 2문장 내 자연 삽입
3. **소제목(H2/H3)** — 2~3개 소제목에 키워드 포함
4. **본문 중간** — 300~500자 간격으로 자연 삽입
5. **마무리** — 마지막 문단에 1회

### 키워드 밀도
- **적정**: 2~3% (1,000자당 20~30자 분량의 키워드)
- **과도**: 4% 이상 → 키워드 스터핑으로 판단될 위험
- **부족**: 1% 미만 → 주제 관련성 약하다고 판단될 수 있음

### 자연스러운 삽입 패턴
```
✓ "일산다이어트한의원을 찾는 분들의 공통 고민은..."
✓ "최근 일산다이어트한의원에서는 체질 기반 접근을..."
✗ "일산다이어트한의원 일산다이어트한의원 일산다이어트한의원..."
```

### 롱테일 키워드 활용
- 메인 키워드의 변형을 자연스럽게 혼용
- 예: "일산다이어트한의원" + "일산 한약 다이어트" + "일산 체질 다이어트"

---

## 3. 소제목 작성

### 규칙
- H2 사용 (H1은 제목에만)
- 3~5개 소제목이 적정
- 키워드를 1~2개 소제목에 포함
- 독자의 궁금증을 반영하는 질문형도 효과적

### 좋은 소제목 예시
```
## 나이가 들수록 다이어트가 어려운 이유
## 일산다이어트한의원에서 먼저 확인하는 것
## 체질 개선 다이어트, 어떤 과정으로 진행되나요?
## 실제로 어떤 변화가 나타나나요?
```

## 4. 문단 작성

### 길이
- 1문단 80~150자 (모바일에서 3~5줄)
- 긴 문단은 독자 이탈 유발
- 핵심 포인트마다 문단 분리

### 톤
- **정보형**: 객관적 사실 + 전문 용어(쉬운 설명 병행)
- **공감형**: 독자 상황 묘사 + 감정적 표현
- 상위글 분석에서 나온 톤 분포를 따라가되, 일관성 유지

### 가독성
- 짧은 문장 선호 (1문장 40자 이내)
- 강조(**볼드**)는 문단당 1~2회
- 번호 리스트(1. 2. 3.)나 불릿(•) 활용

## 5. 이미지 배치

### 규칙
- **최소 3장, 최적 5~8장** (상위글 평균에 맞춤)
- 소제목 직후 또는 핵심 설명 후에 배치
- 이미지 간 텍스트 간격: 200~400자
- 첫 이미지는 2~3문단 이내에 배치

### 이미지 마커 문법
텍스트 작성 시 이미지 위치에 마커를 삽입합니다:
```
[IMAGE: {type} - {description}]
```

**예시:**
```
[IMAGE: real_photo - 한의원 진료실에서 상담하는 장면]
[IMAGE: infographic - 건강 개선 흐름을 나타내는 아이콘과 화살표 레이아웃]
[IMAGE: lifestyle_photo - 건강한 식단을 준비하는 40대 여성]
```

> **주의**: description은 순수 시각적 장면만 묘사합니다. "4단계", "수치", "차트", "표", "번호" 등 텍스트/숫자를 암시하는 표현은 금지합니다. AI 이미지 생성 시 한글·영문 텍스트가 깨지므로 모든 이미지는 텍스트 없이 시각 요소만으로 구성됩니다.

## 6. 네이버 특화 규칙

### 제목
- 30~40자 이내
- 메인 키워드 앞쪽 배치
- 클릭 유도 표현: "알아보기", "확인해야 할", "놓치면 안 되는"

### 태그
- 메인 키워드 + 롱테일 2~3개
- 지역명 포함 (지역 비즈니스인 경우)

### 발행 시간
- 오전 7~9시, 점심 12~1시, 저녁 7~9시가 유리
- 꾸준한 발행 빈도가 C-Rank에 영향

---

## 7. 규제 업종 추가 규칙

의료/금융/법률 업종은 텍스트 원고에서도 컴플라이언스를 준수합니다.

### 의료 업종
- `medical-ad-compliance.md` 참조
- SEO 텍스트는 브랜드 무관한 정보성이지만, 특정 시술/약효 언급 시 주의
- "~에 효과가 있다"보다 "~에 도움이 될 수 있다"
- 수치 언급 시 출처 필요

### 공통
- 결과 보장 표현 금지
- 경쟁사 비방 금지
- 허위/과장 정보 금지
