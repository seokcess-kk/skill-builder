# 네이버 블로그 상위노출 요인

## 주요 랭킹 요인

### C-Rank (Creator Rank)
- 블로그의 전문성과 신뢰도 점수
- 특정 주제 카테고리에서 꾸준히 포스팅할수록 상승
- 주제 이탈(갑자기 다른 분야 글) 시 하락
- 최소 6개월 이상 일관된 주제 포스팅 필요

### DIA (Deep Intent Analysis)
- 검색 의도와 콘텐츠의 매칭도
- 검색어에 대한 답변이 글 안에 있는지 판단
- 소제목, 본문에 검색 의도 반영 필요

### 체류 시간 (Dwell Time)
- 독자가 글에 머무는 시간
- 긴 체류 = 양질의 콘텐츠로 판단
- 이미지, 소제목 구분, 가독성이 체류 시간에 영향

---

## 콘텐츠 요인

| 요인 | 최적값 | 영향도 |
|------|-------|--------|
| 글자수 | 2,000~3,000자 | 높음 |
| 이미지 수 | 5~10장 | 높음 |
| 소제목 | 3~5개 (H2 기준) | 중간 |
| 키워드 밀도 | 2~3% | 높음 |
| 첫 키워드 위치 | 첫 100자 이내 | 높음 |
| 문단 길이 | 80~150자 | 중간 |
| 동영상 포함 | 1개 이상 | 보너스 |
| 링크 (내부/외부) | 1~3개 | 낮음 |

## 상위글 분석 시 확인 항목

경쟁글 분석에서 아래 항목을 체크합니다:

### 구조 분석
- 전체 글 길이 (글자수)
- 소제목 개수와 구성
- 도입-본론-결론 비율
- 문단 수와 평균 문단 길이

### 키워드 분석
- 제목 내 키워드 위치
- 본문 내 키워드 출현 횟수
- 키워드 변형 사용 패턴
- 연관 키워드 포함 여부

### 이미지 분석
- 총 이미지 수
- 이미지 유형 (실사/일러스트/인포그래픽/텍스트카드)
- 이미지 배치 위치 (소제목 후/문단 중간/마무리)
- 이미지 간 텍스트 간격

### 톤/스타일 분석
- 문체 (격식/반말/혼용)
- 정보 전달 방식 (나열형/스토리형/Q&A형)
- 전문가 인용 여부
- CTA 유무 및 위치
