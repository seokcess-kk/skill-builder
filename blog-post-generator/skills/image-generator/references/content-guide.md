# 범용 블로그 콘텐츠 작성 가이드

설득 퍼널 기반 블로그 콘텐츠의 섹션별 텍스트 작성 가이드입니다.
업종에 관계없이 적용 가능한 범용 구조를 제공합니다.

---

## SECTION 01: Hook (오프닝 인용문)

**역할**: 스크롤을 멈추게 하는 첫 번째 장치
**분량**: 1~2문장

### 작성 규칙
- 전문가/대표가 직접 말하는 어투 또는 강렬한 선언문
- 독자의 페인포인트 공감 OR 솔루션 암시 OR 의문 제기
- 지역명/브랜드명 미포함 (범용성 유지)
- 큰따옴표 안에 배치 (인용문형)

### 유형별 예시

**공감형**: "[문제]는 단순히 [표면 현상]이 아닙니다. [근본 원인]부터 바꿔야 합니다"
**희망형**: "[대상]에게 딱 맞는 방법, 분명 있습니다"
**도전형**: "[상황]이라 [행동]이 어렵다고요? 바쁜 일상 속에서도 가능한 방법이 있습니다"
**질문형**: "[실패 경험], 혹시 본인 탓이라고 생각하고 계신가요?"

---

## SECTION 02: Intro (전문가/브랜드 소개)

**역할**: 전문가 신뢰감 또는 브랜드 권위 구축
**분량**: 프로필 + 도입 멘트 3~5문장

### 전문가 인터뷰형
```
INTERVIEW WITH [이름]

[직함/자격]
[소속/브랜드]

"안녕하세요, [브랜드] [이름]입니다."
"오늘은 [주제]에 대해 이야기해보겠습니다."
"특히 [타겟]분들이 가장 궁금해하시는 부분을 중심으로 풀어보겠습니다."
```

### 브랜드 소개형 (전문가 없는 경우)
```
[브랜드명]

[핵심 가치/슬로건]
[설립/경력 요약]

"[브랜드]가 [타겟]을 위해 준비한 이야기입니다."
```

---

## SECTION 03: Pain Point (문제 제기)

**역할**: 독자 공감 ("이거 내 얘기!")
**분량**: 300~500자

### 작성 규칙
- 타겟 페르소나의 구체적 고민 3가지 제시
- 감정적 언어 사용 (좌절, 불안, 피로 등)
- 마지막에 원인 분석으로 전환하는 브릿지 문장

### 형식 옵션

**Voice Card형**: 3개의 독립된 고민 인용 카드
```
"[고민 1 — 감정적 표현]"
"[고민 2 — 구체적 상황]"
"[고민 3 — 반복 실패 경험]"

이분들의 공통점이 있습니다.
[전환 문장 → Why 연결]
```

**Q&A형**: 질문-답변 구조
```
Q. [브랜드]를 찾는 분들의 공통 고민이 있나요?
A. 네, 많은 분들이 비슷한 이야기를 하세요.
[고민 나열]
[전환 문장]
```

**스토리텔링형**: 가상 사례 기반
```
최근 [타겟]분이 찾아오셨습니다.
"[구체적 고민 인용]"
[시도했던 것들 나열]
그런데 왜 번번이 [실패 표현]했을까요?
```

## SECTION 03b / 04b: Bridge (전환)

**역할**: 섹션 간 자연스러운 연결
**분량**: 1~2문장

### 작성 규칙
- Serif 폰트, 중앙 정렬이 기본
- 앞 섹션의 문제의식을 받아 → 다음 섹션의 해결 방향을 암시
- 질문형 또는 선언형

### 예시
- Pain → Why: "반복되는 실패, 그 진짜 원인을 알고 계신가요?"
- Why → Solution: "원인을 알았다면, 이제 방법을 바꿀 차례입니다"

---

## SECTION 04: Why (원인 분석)

**역할**: 전문 지식 기반 원인 설명
**분량**: 400~600자

### 작성 규칙
- 전문 용어는 쉬운 설명 병행: "[용어], 쉽게 말해 [설명]"
- 숫자/통계 적극 활용: "[대상]의 [N]%가 [사실]"
- 비교 구조: 일반적 접근 vs 전문적 접근
- 마지막에 솔루션 연결 전환문

### 구조 옵션

**원인 카드형**: 2~4개 원인을 카드로 분리
```
원인 1: [제목] — [설명]
원인 2: [제목] — [설명]
원인 3: [제목] — [설명]

→ "그래서 [표면적 해결]이 아닌 [근본 해결]이 필요합니다"
```

**타임라인형**: 문제 발생 순서로 설명
**통계 강조형**: 핵심 수치 하나를 크게 배치 + 부연

---

## SECTION 05: Solution (솔루션 제시)

**역할**: 브랜드 차별화 포인트 (전환 핵심)
**분량**: 500~800자

### 작성 규칙
- USP(고유 판매 제안) 중심으로 구성
- 단계형 또는 비교형으로 구조화
- 마지막에 감성적 클로징 문장

### 구조 옵션

**단계형 (타임라인)**:
```
Step 1: [진단/분석] — [설명]
Step 2: [솔루션 적용] — [설명]
Step 3: [관리/유지] — [설명]
Step 4: [결과/확인] — [설명]
```

**비교형**:
```
| 일반적 방법 | [브랜드] 방법 |
|------------|-------------|
| [한계 1]    | [차별점 1]  |
| [한계 2]    | [차별점 2]  |
```

**어프로치 카드형**: USP별 독립 카드로 나열

---

## SECTION 06: Proof (사례/후기)

**역할**: 간접 체험을 통한 신뢰 강화
**분량**: 200~400자

### 작성 규칙
- 규제 업종(의료 등)은 반드시 컴플라이언스 가이드 준수
- 수치 사용 시 출처/기준 명시 + 개인차 고지
- 과장 표현 금지

### 구조 옵션

**수치 강조형**:
```
[브랜드] 실적:
• [수치 1] — [설명]
• [수치 2] — [설명]
• [수치 3] — [설명]
* [출처/기준], 개인별 결과는 차이가 있을 수 있습니다.
```

**후기 인용형**:
```
"[고객 후기 인용]"
— [고객 속성, 예: 30대 직장인]

* 개인의 경험이며, 결과는 개인에 따라 다를 수 있습니다.
```

**전문가 코멘트형**:
```
Q. 실제 반응은 어떤가요?
A. 가장 많이 들은 말씀이 "[대표 후기]"입니다.
특히 [타겟]분들이 놀라시는 부분은 [구체적 장점]입니다.
```

## SECTION 07: CTA (클로징)

**역할**: 내원/구매/상담 전환 유도
**분량**: 브랜딩 문구 + 연락처

### 작성 규칙
- 행동 유도는 1가지로 명확하게
- 브랜드 슬로건 또는 감성 클로징 문구
- 연락처/예약 방법 포함

### 클로징 문구 유형
- **변화형**: "[문제]에서 자유로워지는 그날까지"
- **시작형**: "오늘이 마지막 [시도]가 될 수 있도록"
- **초대형**: "[브랜드]에서 시작하세요"
- **약속형**: "[타겟]을 위한 [브랜드]의 약속"

### 구조
```
[영문 브랜드 라벨]

[브랜딩 클로징 문구]

[브랜드명] [지점]
☎ [전화번호]
📍 [주소]
[예약/상담 채널 안내]
```

---

## SECTION 10: Disclaimer (법적 고지)

**역할**: 법적 보호 + 신뢰성 보완
**분량**: 업종별 필수 고지 항목에 따라 자동

업종별 상세 고지 내용:
- 의료 → `medical-ad-compliance.md` 4장 참조
- 금융 → 원금 손실 가능성, 과거 수익률 미보장
- 법률 → 법적 조언 아님, 개별 상담 권고
- 일반 → 결과 개인차, 정보성 콘텐츠 안내

## 키워드 삽입 가이드

### 삽입 위치
1. Pain Point 질문부에 2~3회
2. 소제목/배너에 1~2회
3. 본문 중간에 2~3회
4. CTA에 1회
5. 이미지 파일명에 포함

### 키워드 밀도
- 전체 텍스트의 2~3% (자연스러운 문맥 내 삽입)
- 키워드 스터핑 금지
