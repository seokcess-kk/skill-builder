# 다이트한의원 블로그 콘텐츠 템플릿

7개 섹션 구조의 인터뷰형 블로그 콘텐츠 작성 가이드입니다.

**브랜드명**: 다이트한의원 / **영문**: DAEATDIET
- CTA 상단 라벨: `DAEATDIET`
- disclaimer 푸터: `DAEATDIET`
- "Diet Korean Medicine" 등 다른 영문 표기 사용 금지

## SECTION 01: 오프닝 인용문 (Hook)

**역할**: 스크롤을 멈추게 하는 첫 번째 장치
**분량**: 1~2문장

### 작성 규칙
- 원장님이 직접 말하는 어투 ("~입니다", "~합니다")
- 환자의 페인포인트 공감형 OR 솔루션 암시 희망형
- 지역명 미포함 (범용성 유지)
- 큰따옴표 안에 배치

### 유형별 예시

**공감형**: "다이어트는 단순히 체중을 줄이는 게 아닙니다. 몸이 스스로 건강한 체중을 유지할 수 있도록 체질부터 바꿔야 합니다"

**희망형**: "나이가 들수록 빠지지 않는 살, 방법이 아닌 원인부터 찾아야 합니다"

**도전형**: "직장인이라 다이어트가 어렵다고요? 바쁜 일상 속에서도 가능한 방법이 있습니다"

**질문형**: "다이어트에 실패한 경험, 혹시 본인 탓이라고 생각하고 계신가요?"

## SECTION 02: 원장 소개 (Interview Intro)

**역할**: 전문가 신뢰감 구축
**분량**: 프로필 + 도입 멘트 3~5문장

### 작성 규칙
- "INTERVIEW" 또는 "INTERVIEW WITH [원장명]" 타이틀
- 프로필: 이름, 직함, 전문 분야
- 도입 멘트: 인사 + 오늘 주제 소개

### 템플릿

```
INTERVIEW
[원장명] 원장

다이트한의원 [지역] 대표원장
한방 비만 전문의 / [전문 분야]

---

"안녕하세요, 다이트한의원 [지역]점 [원장명]입니다."

"오늘은 많은 분들이 궁금해하시는 [주제]에 대해 
자세히 이야기해보겠습니다."

"특히 [타겟 페르소나]분들이 가장 많이 하시는 질문을 
중심으로 풀어보겠습니다."
```

## SECTION 03: 문제 제기 (Pain Point)

**역할**: 독자 공감 유도 ("이거 내 얘기!")
**분량**: 300~500자

### 페르소나별 페인포인트

| 타겟 | 핵심 고민 | 예시 문구 |
|------|-----------|-----------|
| 30~40대 직장인 | 시간 부족, 야식, 회식 | "퇴근하면 녹초인데 운동까지?" |
| 40~50대 중년 | 대사 저하, 요요 | "예전엔 굶으면 빠졌는데 이제는..." |
| 20~30대 여성 | 반복 실패 | "다 해봤는데 왜 저만 안 빠질까요?" |
| 산후 여성 | 체형 변화 | "출산 후 몸이 완전히 달라졌어요" |
| 건강 고민 중년 | 혈압/혈당/콜레스테롤 | "살도 빼야 하는데 건강 수치도 걱정" |

### 형식 옵션

**Q&A 형식 (추천)**:
```
Q. [지역] 다이어트한의원을 찾는 분들의 공통적인 고민이 있나요?

A. 네, 많은 분들이 비슷한 이야기를 하세요.
"[페인포인트 1]"
"[페인포인트 2]"

사실 이분들의 공통점이 있습니다.
[전환 문장 → 다음 섹션 연결]
```

**스토리텔링 형식**:
```
최근 [지역]점에 [타겟] 분이 오셨습니다.
"원장님, 저 다이어트 진짜 다 해봤어요."

[구체적 시도 나열]
그런데 왜 번번이 실패했을까요?
그 이유를 알면 마지막 다이어트가 가능합니다.
```

## SECTION 04: 원인 분석 (Why)

**역할**: 전문 지식 기반 실패 원인 설명
**분량**: 400~600자

### 활용 가능한 원인 프레임

| 원인 | 핵심 설명 |
|------|-----------|
| 기초대사량 저하 | 나이/근육량 감소 → 같은 양 먹어도 축적 |
| 체질 불균형 | 습담, 어혈, 기허 → 체질별 반응 차이 |
| 호르몬 변화 | 갑상선, 인슐린 저항성, 여성호르몬 |
| 생활 패턴 | 수면부족, 스트레스, 불규칙 식사 |
| 잘못된 다이어트 이력 | 극단적 제한 → 요요 → 체지방 증가 악순환 |

### 작성 규칙
- 의학 용어는 쉬운 설명 병행: "습담, 쉽게 말해 몸속 노폐물"
- 숫자 활용: "40대 이후 기초대사량은 매년 1~2% 감소"
- 비교 구조 활용: 일반 다이어트 vs 한의원 접근
- 마지막에 솔루션 연결 전환문: "그래서 체중이 아닌 체질을 바꿔야 합니다"

### 템플릿

```
[소제목 배너: "반복되는 실패의 공통점" / "나이가 들수록 어려워지는 진짜 이유"]

많은 분들이 [일반적 방법]을 시도합니다.
하지만 효과가 없었던 이유는 [원인 1]과 [원인 2] 때문입니다.

우리 몸은 [한의학적 설명 1~2문장].
특히 [타겟층]의 경우 [구체적 설명]이 중요합니다.

[전환: "그래서 체중이 아닌 체질을 바꿔야 합니다"]
```

## SECTION 05: 솔루션 제시 (How)

**역할**: 한의원 차별화 포인트 (전환 핵심)
**분량**: 500~800자

### USP 풀

| USP | 설명 |
|-----|------|
| 1:1 맞춤 처방 | 개인별 체질 분석 후 한약 조제 |
| 체질 진단 시스템 | 사상체질 + 현대 진단 결합 |
| 단계별 관리 | 진단→처방→관리→유지 4단계 |
| 생활습관 교정 | 식이요법, 운동 가이드 병행 |
| 요요 방지 | 감량 후 유지 단계까지 관리 |
| 직장인 특화 | 간편 복용, 짧은 진료 시간 |
| 원장 직접 소통 | 1:1 상담, 중간 체크 |

### 형식 옵션

**비교형 (추천)**:
```
[소제목 배너: "[지역] 다이어트한의원의 접근 방식"]

[기존 방법 한계 1~2문장]
다이트한의원에서는 다릅니다.

① 체질 진단: [설명]
② 맞춤 한약 처방: [설명]  
③ 생활습관 교정: [설명]
④ 유지 관리: [설명]

"단순히 살을 빼는 것이 아니라,
건강한 체중을 유지할 수 있는 몸을 만드는 것이 목표입니다."
```

**스토리 연결형**:
```
[소제목 배너: "마지막 다이어트가 될 수 있도록"]

앞서 말씀드린 [환자 사례]의 경우,
[진단 결과]였습니다.
[처방 방향]으로 접근해 [기간] 만에 [결과]를 보셨습니다.

핵심은 [차별화 포인트]입니다.
```

## SECTION 06: 환자 사례/후기 (Social Proof)

**역할**: 간접 체험을 통한 신뢰 강화
**분량**: 200~400자

### ⚠️ 의료광고법 주의사항
- 구체적 감량 수치 → "개인 차이가 있을 수 있음" 고지 필수
- 비포/애프터 사진 → 광고심의 기준 확인
- "100% 성공", "무조건" 등 과장 표현 금지
- 환자 동의 없는 사례 공개 금지

### 형식 옵션

**원장 코멘트형 (안전)**:
```
Q. 실제 환자분들의 반응은 어떤가요?

A. 가장 많이 들은 말씀이
"이렇게 편한 다이어트는 처음이에요"입니다.

특히 [타겟층] 분들이 놀라시는 부분은
[구체적 장점]이에요.

* 치료 효과는 개인에 따라 차이가 있을 수 있습니다.
```

**숫자 강조형**:
```
다이트한의원 [지역]점:

• 월 평균 상담 [N]건 이상
• 재방문율 [N]%
• 평균 관리 기간 [N]개월

* 내부 데이터 기준, 개인별 결과는 차이가 있을 수 있습니다.
```
